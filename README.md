# 🎬 Netflix Content Strategy — EDA Project

## 🔍 What Problem Did I Solve?
Netflix has 8,800+ titles — but is the platform 
shifting from Movies to TV Shows? I used Python 
to find out.

## 📊 What I Found
| Finding | Detail |
|---|---|
| Content peaked | 2019 — dropped post COVID |
| TV Shows trend | Added more recently than Movies (t-test validated) |
| Top genre | Drama + International Movies |
| International growth | Rising every year since 2016 |
| Dominant rating | TV-MA — adult-focused platform |

## 🔬 Hypothesis I Tested
**Are TV Shows newer additions than Movies on Netflix?**  
✅ Yes — statistically proven (p < 0.05, Independent T-Test)
